# Product Vision: Cyber-Dock

> "The dock iDRAC would be if Dell made a USB-C dock."

## Competitive Edge
Unlike standard Thunderbolt/USB-C docks (CalDigit, Anker), the Cyber-Dock is an **autonomous server**.
- **Remote BIOS Access:** Via PiKVM, even when the laptop is off or OS is crashed.
- **Tailscale/WireGuard:** Secure remote access to the dock and the connected laptop's local network.
- **ISO Library:** Mount ISOs over USB as virtual CD-ROMs for OS installs.
- **24/7 Resilience:** Independent power and networking.

## Feature Matrix
- **Must-Have:** 100W PD, PiKVM, NVMe NAS, dual 2.5GbE.
- **Stretch:** OLED display, audio DAC, front microSD.
- **v2:** 4K capture, USB4, PoE support.
